1. مقدمة ونطاق العمل السيادي

يقدم هذا التقرير الهندسي السيادي الاستيعاب الكامل للمعادلات الإنشائية الحاكمة لمنصة المدقق الديناميكي الموحد V3.0، المستخلصة من الأطروحة العلمية للتحصين وملفات الإكسيل المرجعية السبع. يغطي التقرير جميع المعادلات الخاصة بحسابات ضغوط العصف الديناميكي، والتحقق من وقوع محصلة الأحمال داخل نواة المقطع، وفحص السعة وإجهادات القص الثاقب وفق الكود العربي السوري لعام 2024 بطريقة إجهادات العمل (WSD) ومعايير UFC 3-340-02 الصادرة عن وزارة الدفاع الأمريكية.

تم تحويل جميع المنحنيات والرسومات البيانية الموجودة في الأطروحة والإكسيل إلى دوال استيفاء رياضي رقمية مبرمجة (Interpolation Functions) لاعتمادها كـ Baseline Constants غير قابلة للتعديل اليدوي، مما يضمن نسبة انحراف 0.00% عند ربطها حركياً بالبنية التحتية للمنصة (Next.js PWA / TypeScript) في الخطوة التالية.

2. المعادلات الأساسية لضغوط العصف الديناميكي

2.1 معادلات سادوفسكي للضغط الزائد (Sadovsky Overpressure Formulas)

تُعد معادلات سادوفسكي الحجر الأساس في حساب الضغوط الناتجة عن الانفجارات، وهي مستخدمة في الكود العربي السوري 2024 ومعايير UFC 3-340-02 كمرجعية أساسية. تشمل هذه المعادلات ثلاث حالات رئيسية: الضغط الساقط العمودي، والضغط المنعكس، والضغط عند زاوية سقوط مائلة.

المعادلة (1): الضغط الزائد الساقط - Sadovsky Incident Overpressure
ΔP = 0.1 · ∛C / R + 0.43 · ∛C² / R² + 1.4 · C / R³
المعادلة (2): الضغط الزائد المنعكس - Reflected Overpressure
ΔPφ = 0.084 · ∛C / R + 0.27 · ∛C² / R² + 0.7 · C / R³
المعادلة (3): الضغط عند زاوية مائلة - Oblique Incidence Overpressure
ΔPφ = 0.146 · √(C/(S·R)) + 0.92 · √(C²/(S²·R²)) + 4.4 · C/(S·R)

حيث: C = كمية المتفجرات (كجم)، R = المسافة بين مركز الانفجار ونقطة القياس (م)، S = مساحة السطح المتأثر (م²)، φ = زاوية سقوط الموجة. هذه المعادلات صالحة عندما تكون النسبة √C·√R/T أقل من أو تساوي 1000، حيث T = 2π/ω هو دور الاهتزاز الطبيعي للمنشأ.

2.2 معادلات الطور الموجب والاندفاع (Positive Phase & Impulse)

يتم حساب زمن الطور الموجب والاندفاع الناتج عن الانفجار بعدة طرق، والأكثر استخداماً في التصميم الحمائي هي الطريقة الأولى لسادوفسكي.

المعادلة (4): زمن الطور الموجب - الطريقة الأولى (سادوفسكي)
τ+ = 1.7 × 10-3 · ∛√C · √R
المعادلة (5): الاندفاع - الطريقة الأولى
i = 6.3 · ∛C² / R
المعادلة (6): زمن الطور الموجب - التفجير السطحي
τ+ = 1.5 × 10-3 · ∛√C · √R
المعادلة (7): الاندثار - التفجير السطحي
i = 4 · ∛C² / R
المعادلة (8): الزمن الفعال للموجة
τэф = τ+ · (0.0008 · (ΔPmax)² - 0.0384 · ΔPmax + 1.0013)

المعادلة (8) هي معادلة استيفاء رقمية مبرمجة مشتقة من المنحنيات التجريبية في الأطروحة، وتُعد من Baseline Constants الأساسية للمنصة.

2.3 معادلات الضغط المنعكس عند السطح (Reflected Pressure)

المعادلة (9): الضغط المنعكس مع اقتران السرعة-الضغط
ΔPi(t,z) = ki · ΔP(t,z) - ρ · ai · Vi

حيث ki = معامل الانعكاس (يعتمد على زاوية السقوط ونوع السطح)، ρ = كثافة الوسط، ai = سرعة الموجة في الوسط، Vi = السرعة الفعلية للوسط عند نقطة الاصطدام. عند تجاهل تأثير السرعة: a/ai = 2 (معامل الانعكاس للسطح الصلب).

المعادلة (14): معامل تأثير شكل الرأس الحربي

المعادلة (15): معامل تأثير القطر

5. التحقق من وقوع محصلة الأحمال داخل نواة المقطع

5.1 شرط النواة للمقطع الخرساني (Section Core Condition)

يُعد التحقق من وقوع محصلة الأحمال داخل نواة المقطع الخرساني شرطاً أساسياً في طريقة إجهادات العمل (WSD) وفق الكود العربي السوري 2024، لضمان عدم حدوث إجهادات شد في المقطع الخرساني تحت تأثير الأحمال الديناميكية. تقع النواة في المقطع المستطيل ضمن الثلث الأوسط من كل بعد، ويجب أن تكون اللامركزية أقل من أو تساوي سدس البعد الكلي للمقطع.

المعادلة (38): شرط النواة للمقطع المستطيل
e ≤ h/6 ⟹ محصلة الأحمال داخل النواة (لا إجهاد شد)
المعادلة (39): اللامركزية
e = M / N
المعادلة (40): العزم على السقف (تكديس)
Ms = Pp · ap² / 12 (عزوم تكديس)
المعادلة (41): العزم على السقف (مفصلي)
Ms = Pp · ap² / 8 (عزوم مفصلية)

5.2 التحقق الرقمي للحالات المرجعية

بالاعتماد على بيانات ملف حساب حمولة الانفجار على سقف النفق، نتحقق من شرط النواة:

التحقق - حالة سقف النفق:
• الحمل التصميمي: Pp = 1.9 kg/cm² = 19 t/m²
• البعد القصير: ap = 1.0 m
• البعد الطويل: bp = 3.0 m
• سماكة السقف المعتمدة: Hп = 40 cm = 0.40 m
• العزم المفصلي: Ms = 24 t·m (معتمد من حسابات الإكسيل)
• القوة المحورية: N = Pp · bp = 19 × 3 = 57 t/m
• اللامركزية: e = M/N = 24/57 = 0.421 m
• حد النواة: h/6 = 0.40/6 = 0.067 m
• النتيجة: e = 0.421 > h/6 = 0.067 ⟹ محصلة الأحمال خارج النواة
• ملاحظة: هذا شرط متوقع في المنشآت الحمائية المعرضة لأحمال انفجار ديناميكية كبيرة، ويتطلب تصميم تسليح مزدوج للشد.
توصية هندسية: في المنشآت الحمائية حيث تخرج محصلة الأحمال عن نواة المقطع بشكل متوقع بسبب الأحمال الديناميكية العالية، يجب استخدام معادلات المقطع الخرساني المصحح مع تسليح مزدوج (فقاعات شد وضغط) وفق الفصل الثامن من الكود العربي السوري 2024. يجب التحقق من: σc ≤ Rb,н/Kн و σs ≤ Ra,н/Kн حيث Kн معامل الأمان.

6. فحص إجهادات القص الثاقب (Punching Shear)

6.1 معادلات القص الثاقب وفق الكود العربي السوري 2024 (WSD)

يتم فحص القص الثاقب عند الأعمدة والجدران الحاملة في المنشآت الحمائية للتأكد من أن إجهاد القص المحيطي لا يتجاوز مقاومة القص للخرسانة. في طريقة إجهادات العمل (WSD)، يتم استخدام معاملات أمان مناسبة للأحمال الديناميكية.

المعادلة (42): إجهاد القص الثاقب
τp = P / (u · h₀)
المعادلة (43): المحيط الحرج للقص الثاقب
u = 2·(a + h₀) + 2·(b + h₀) (عمود مستطيل)
المعادلة (44): مقاومة القص الثاقب للخرسانة - WSD
τcr = Kн · Rbt (للأحمال الساكنة) τcr,d = DIF · Kн · Rbt (للأحمال الديناميكية)

حيث: P = القوة القاصمة (كجم)، u = المحيط الحرج (سم)، h₀ = العمق الفعال (سم)، Rbt = مقاومة شد الخرسانة، DIF = معامل الزيادة الديناميكية، Kн = معامل الأمان.

6.2 معايير UFC 3-340-02 للقص الثاقب الديناميكي

وفق معايير UFC 3-340-02 الفصل الرابع، يتم حساب القص الثاقب مع مراعاة التأثير الديناميكي للانفجار كالتالي:

المعادلة (45): مقاومة القص الثاقب وفق UFC 3-340-02
Vn = (2 + 4/βc) · √f'dc · b₀ · d
المعادلة (46): مقاومة الضغط الديناميكية للخرسانة
f'dc = f'c · DIF
المعادلة (47): معامل الزيادة الديناميكية - UFC 3-340-02
DIFconc = (ẋ / ẋ₀)1/(1+0.6·log₁₀(ẋ/ẋ₀)) (للخرسانة)

حيث: βc = نسبة جانب العمود الطويل إلى القصير، f'dc = مقاومة الضغط الديناميكية، b₀ = محيط المحور الحرج، d = العمق الفعال، ẋ = معدل التشوه، ẋ₀ = معدل التشوه المرجعي.

6.3 التحقق الرقمي من القص الثاقب

التحقق - حالة سقف النفق (عمود/جدار داعم):
• سماكة السقف: h = 40 cm، h₀ = 35 cm
• الخرسانة: M350 → Rb,н = 200 kg/cm²
• مقاومة الشد: Rbt ≈ 12.5 kg/cm²
• DIF للقص = 1.25 (وفق UFC 3-340-02 Table 4-1)
• τcr,d = 1.25 × 12.5 = 15.6 kg/cm²
• الحمل التصميمي: Pp = 1.9 kg/cm² على مساحة 3.0 × 1.0 m
• القوة القاصمة عند جدار داعم عرض 30 cm: P ≈ 1.9 × 100 × 30 = 5,700 kg
• المحيط الحرج: u = 2(30+35) + 2(100+35) = 400 cm
• τp = 5700 / (400 × 35) = 0.41 kg/cm²
• النتيجة: τp = 0.41 < τcr,d = 15.6 ⟹ آمن مقابل القص الثاقب ✓
التحقق - حالة مقر القنبلة الخارقة:
• الحمل التصميمي على السقف: Ppп = 4.921 kg/cm²
• سماكة السقف المحسوبة: Hп = 70.46 cm
• h₀ ≈ 65 cm
• القوة القاصمة عند جدار عرض 40 cm، طول 300 cm:
• P ≈ 4.921 × 300 × 40 = 59,052 kg
• المحيط الحرج: u = 2(40+65) + 2(300+65) = 810 cm
• τp = 59052 / (810 × 65) = 1.12 kg/cm²

7.3 دوال الاستيفاء الهندسية

الدالة I-8: حساب هندسة القوس الحمائي (Protodiakonov Method)

مصدر: الأطروحة - الفصل الثاني + سماكة مقطع الجزء الديناميكي

سماكة الغطاء
d₀ = 0.06 · √((2l₀/h₀) · (1 + 2l₀/f))
سماكة السقف
dn = (1.25 ~ 1.5) · d₀
سماكة الجدار
dc = (1 ~ 2) · dn
نصف قطر القوس
R² = (2a/2)² + (R - h₀)² ⟹ R = (a² + h₀²) / (2h₀)
طول القوس
l = θ · R ; tan(θ/2) = a / (R - h₀)
مساحة المقطع
S = ½ · θ · R²

حيث: f = معامل بروتودياكونوف (للصخر الكلسي: f = 6)، 2a = عرض النفق، h₀ = ارتفاع القوس.

الدالة I-9: الزمن الفعال للموجة (تجريبية)

مصدر: ملفات الإكسيل - معادلة الاستيفاء من المنحنيات

معادلة الاستيفاء المبرمجة
τэф = τ+ · f(ΔPmax)
دالة التصحيح المتعددة الحدود
f(ΔP) = 0.0008 · ΔP² - 0.0384 · ΔP + 1.0013

هذه المعادلة مشتقة من المنحنيات التجريبية وتُعد Baseline Constant. مجال الصلاحية: 0 < ΔP < 25 kg/cm².

الدالة I-10: تباعد الأنفاق المتجاورة

مصدر: ملف حساب التباعد بين نفقين متجاورين

التباعد من تأثير القنبلة
Lbomb = 2 · √((1.75·Zp)² - (H - hпр + ц)²) - 2a
التباعد من ضغط الصخر
Lrock = 0.65 · √(a · H · γ / fkp)
عمق المنطقة المتأثرة
Zp = 1.5 · kp.г · ∛Cэф
ارتفاع حد المنطقة الديناميكية
H = R̽ + hпр - ц

التباعد المطلوب: L ≥ max(Lbomb, Lrock)

النتائج المرجعية: Lbomb = 19.652 m, Lrock = 2.206 m → التباعد المعتمد: 19.652 m

// I-9: Effective time interpolation
function effectiveTime(
  tauPlus: number, deltaPmax: number
): number {
  const f = 0.0008 * deltaPmax**2
          - 0.0384 * deltaPmax
          + 1.0013;
  return tauPlus * f;
}

// I-10: Tunnel spacing calculation
function tunnelSpacing(
  Zp: number, H: number, hpr: number,
  tsu: number, a: number, gamma: number,
  fkp: number
): { L_bomb: number; L_rock: number } {
  const L_bomb = 2 * Math.sqrt(
    (1.75 * Zp)**2 - (H - hpr + tsu)**2
  ) - 2 * a;
  const L_rock = 0.65 * Math.sqrt(
    a * H * gamma / fkp
  );
  return { L_bomb, L_rock };
}
